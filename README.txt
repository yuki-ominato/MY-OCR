# OCR文字認識アプリケーション

## 概要
このアプリケーションは、画像から文字を認識するOCR（光学文字認識）システムです。ディープラーニングを使用して、手書き文字や印刷文字を高精度で認識することができます。

## 主な機能
- 画像の前処理（グレースケール変換、ノイズ除去、二値化）
- データ拡張による学習データの強化
- CNN（畳み込みニューラルネットワーク）による文字認識
- 学習済みモデルの保存と再利用

## 技術スタック
- Python 3.x
- TensorFlow/Keras
- OpenCV
- NumPy
- Pandas

## 必要なライブラリ
```
tensorflow
keras
opencv-python
numpy
pandas
pillow
```

## 使用方法

### 1. 環境セットアップ
必要なライブラリをインストールします：
```bash
pip install -r requirements.txt
```

### 2. データの準備
- 学習データはCSVファイル（train.csv）で提供する必要があります
- CSVファイルには以下の列が必要です：
  - Label: 認識対象の文字
  - Pixel Values: 画像のピクセル値（スペース区切りの数値）

### 3. モデルの学習
```bash
python model.py
```
- 学習済みモデルは `saved_models/ocr_model.keras` に保存されます
- 前処理済みの画像は `preprocessed_train_images/` ディレクトリに保存されます

## モデルの構造
- 入力層: 32x32x1（グレースケール画像）
- 3つの畳み込みブロック（各ブロックに2つの畳み込み層）
- バッチ正規化とドロップアウトによる過学習防止
- 全結合層による分類

## データ拡張
学習時に以下のデータ拡張を適用：
- 回転（±10度）
- シフト（幅・高さの10%）
- ズーム（±10%）

## 注意事項
- 学習には十分なGPUメモリが必要です
- 大量のデータを処理する場合は、メモリ使用量に注意してください
- 学習時間はデータ量とハードウェアの性能に依存します
